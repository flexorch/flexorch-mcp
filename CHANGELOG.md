# Changelog

## [Unreleased]

### Added
- `process_document` tool — upload and process documents via FlexOrch pipeline
- `get_job_status` tool — poll job status until completed or failed
- `get_extraction_result` tool — retrieve structured extracted fields
- `build_dataset` tool — build a dataset from a completed execution
- `search_documents` tool — semantic search across indexed datasets (Pro+)
- `export_dataset` tool — export datasets in 8 formats (JSONL, CSV, JSON, XML, XLSX, MD, RAG, HuggingFace)
- `flexorch-mcp --check` CLI command for connection verification
- Claude Desktop and Cursor configuration examples
